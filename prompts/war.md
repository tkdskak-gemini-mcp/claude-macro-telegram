## 🚨 데이터 무결성 경량 룰 (2026-05-14 추가)

**핵심 목적 우선**: 선행지표로 전쟁 발생 가능성 포착 → 옵션 배팅 타이밍. 따라서 핵심 선행지표는 그대로 유지하되, **검증 불가 영역에서만 추정 금지**.

### 검증 등급별 처리 룰

**🟢 [A] MCP 직접 호출 가능 — 그대로 사용**
- FRED: WTI(DCOILWTICO), HH 가스(DHHNGSP), DGS10, DXY
- Alpha Vantage/Yahoo: USO·UNG·UVXY·SPY·GLD·SMH·FXI·VST 시세

**🟡 [B] WebSearch 보도자료 검증 — 출처 라벨 부착 시 사용**
- Lloyd's 호르무즈 프리미엄 → "[Lloyd's]" 라벨
- 주권 CDS (이스라엘·대만·파키스탄) → "[보도]" 라벨
- TTF·JKM 가스 → "[ICE]" 또는 "[Platts]" 라벨
- 호르무즈 일일 통항 → "[Bloomberg]" 또는 "[Lloyd's List]"
- VLCC/Suezmax 용선료 → "[보도]" 라벨

**🟠 [C] WebSearch 정성 확인만 — 정성 표현으로만 사용**
- Polymarket 확률 → **마켓 부재 시 "해당 마켓 없음" 명시**, 확률 추정 생성 금지
- GDELT 긴장지수 → **구체 z-score 추정 생성 금지**, "긴장 고조"·"평년 대비" 정성 표현만
- OSINT 위성 이미지 → 보도 인용된 사실만, 직접 분석 추정 금지
- 인터넷 차단 → NetBlocks 보도 사실만

**🔴 [D] 검증 불가능 — 추정치 생성 절대 금지**
- 펜타곤 피자지수(PizzINT) → **정량 멘션 카운트 생성 금지**. "뉴스 보도 X건" 식으로만
- ADS-B 정확한 비행 시각·고도 → 보도 인용 없으면 "[D] 미확보"
- 특정 CDS의 정확한 bp → 보도 부재 시 "[D] 미확보"

### 트리거 점등 검증 룰

```
1순위 트리거 (6개): [B] 이상 검증 필수
  - 보도자료·FRED·시세 출처가 명확한 경우만 점등 인정
  - 정성 추측만으로 점등 금지

2순위 트리거 (15개): 
  - [B] 이상 검증 → 1.0× 가중치
  - [C] 정성만 → 0.5× 가중치 (절반 점등)
  - [D] 미확보 → 점등 불가, 공란

✅ 점등 표기 예시:
  ✅ TTF €68 [Platts] — 1순위 점등
  🟡 GDELT US-Iran 긴장 고조 [C, 정성] — 2순위 0.5점등
  ⬜ E-6 Mercury 이륙 [D, 미확보] — 점등 불가
```
